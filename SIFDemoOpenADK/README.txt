# Summary
#########################################################################################################
The main purpose of the SIFDemo Project is to give SIF agent developers a small SIF project they can run 
and play with on their local environment and it illustrates how the SIFCommon Framework can be utilised 
to quickly develop an agent and wire all the components up.

For details please refer to the Developer's Guide in the 'UserGuide' directory of this project.


#########################################################################################################
# Download Instructions
#########################################################################################################

How to download this project:

Option 1 - As a Zip.
====================
Click on the button marked "ZIP" available from the Code tab.


Option 2 - Using a Git client.
==============================
From the command-line type: git clone git://github.com/nsip/SIFDemoOpenADK-Java.git

Note that if you want to use this option but don't have the client installed, it can be 
downloaded from http://git-scm.com/download.
